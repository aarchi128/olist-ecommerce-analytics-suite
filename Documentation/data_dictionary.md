# Data Dictionary: Reporting Schema View Definitions (MySQL)

This document details the schema definitions for the core analytics views exposed to Power BI from the MySQL database engine.

---

## 📋 View: `view_delivery_performance`
Provides full lifecycle tracking for order processing, carrier dispatch, and customer door-to-door transit times.

| Column Name | Data Type | Key Reference | Business Definition |
| :--- | :--- | :--- | :--- |
| `order_id` | `VARCHAR(50)` | Primary Key (View) | Unique alphanumeric string identifying an individual transaction. |
| `delivery_status` | `VARCHAR(20)` | None | Current operational status of the parcel (`Delivered`, `Shipped`, `Delayed`, `Canceled`). |
| `days_to_ship` | `INT` | None | **Internal SLA:** The number of days elapsed from initial order timestamp to carrier pickup. |
| `transit_days` | `INT` | None | **External SLA:** The number of days elapsed from carrier pickup to final delivery. |
| `total_actual_delivery_days` | `INT` | None | **End-to-End Cycle:** Total days elapsed from order timestamp to doorstep delivery. |
| `sla_deviation_days` | `INT` | None | Variance against promised delivery timeline. Positive integers signify days past SLA; negative values signify early arrivals. |

---

## 📋 View: `view_customer_rfm_segments`
Maintains behavioral cohort classification tracking updated consumer metrics.

| Column Name | Data Type | Key Reference | Business Definition |
| :--- | :--- | :--- | :--- |
| `customer_id` | `VARCHAR(50)` | Primary Key (View) | Unique identifier mapping to the master consumer profile database. |
| `recency_score` | `INT` | None | Ranked tier (1-5) evaluating the days elapsed since the client's last order. |
| `frequency_score` | `INT` | None | Ranked tier (1-5) evaluating the lifetime order velocity of the client. |
| `monetary_score` | `INT` | None | Ranked tier (1-5) evaluating total historical gross revenue generated by the client. |
| `customer_segment` | `VARCHAR(30)` | None | Algorithmic behavioral categorization assigned based on aggregated RFM scoring combinations. |
